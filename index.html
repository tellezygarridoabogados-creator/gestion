📋 Instrucciones Apps Script
Paso a paso para activar Google Sheets:

1. Ve a script.google.com → Nuevo proyecto
2. Pega el código del archivo apps-script-backend.js adjunto
3. Configura la variable SHEET_ID con el ID de tu Google Sheet
4. Ve a Implementar → Nueva implementación → Aplicación web
5. Ejecutar como: Yo · Acceso: Cualquier persona
6. Copia la URL generada y pégala arriba

Google Sheets: Crea una hoja llamada Causas con las columnas en el orden indicado en el código.
